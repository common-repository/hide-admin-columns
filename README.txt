Hide columns in the WordPress admin list tables for custom post types.

## Description

Do you find the WordPress admin list tables cluttered with too many columns that you rarely use? **Hide Admin Columns** is here to rescue you from the chaos! This powerful plugin gives administrators the ability to hide specific columns in the admin list tables for custom post types, resulting in a cleaner, more focused view.

**Why would you need to hide columns added by other plugins?**

- **Enhanced Focus:** By hiding unnecessary columns, you can focus on the essential data that matters most to your workflow.
- **Improved Performance:** Reducing the number of columns can speed up the loading time of your admin list tables.
- **Clarity and Usability:** A streamlined list table improves readability and makes it easier to manage your content.

With **Hide Admin Columns**, you can keep the valuable functionality of various plugins while hiding their extra columns that clutter your view. This ensures you maintain a clean and efficient workspace without compromising on the features provided by your plugins.

## Installation

1. Upload the plugin files to the `/wp-content/plugins/hide-admin-columns` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Configure the plugin by navigating to **Settings > Hide Admin Columns**.

## Frequently Asked Questions

### How do I hide columns for a specific post type?

Navigate to **Settings > Hide Admin Columns** and select the columns you want to hide for each post type.

### Who can manage the column visibility settings?

Only users with the 'manage_options' capability can modify the column visibility settings.
